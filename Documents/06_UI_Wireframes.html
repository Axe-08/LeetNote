class Solution:
  def twoSum(self, nums, target):
    seen = {}
    for i, n in enumerate(nums):
      if target - n in seen:
        return [seen[target-n], i]
      seen[n] = i

class Solution:
    def twoSum(self, nums, target):
        seen = {}
        for i, n in enumerate(nums):
            if target - n in seen:
                return [seen[target-n], i]
            seen[n] = i
Time  O(n)
Space  O(n)
My Notes
Hash map stores seen values. On each iteration check if complement already exists. Single pass makes it O(n) rather than the naive O(n²) brute force.
Community Insights
"Use a sliding window to track frequency — the O(n) solution is elegant once you realize the hash stores future lookups." — @user23451
